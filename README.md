### ‚ö° Technical Philosophy
*"Code is law"* is a fallacy. Code is a **constitutional draft**. My work focuses on:
1. **Sovereignty**  
   Digital identity should be owned, not rented. Users should be able to exist, transact, and exit without platform dependence or hidden custodianship.
2. **Mechanism Design**  
   Protocols are economic and political systems. Their incentive structures determine whether they serve users or concentrate power. Design choices are never neutral.
3. **Democratic Ownership**  
   A system is only worth building if it is ultimately governed by its users. If people cannot understand it, influence it, or leave it without penalty, then it is not decentralized ‚Äî it is merely automated.
